# Практические задания 2

Форматированные в проект практические задания по психодидактике

## Установка

### Git
Откройте терминал (консоль) в нужной директории (папке) и склонируйте проект (должен быть установлен git):
```bash
git clone https://github.com/catlilface/psychodid
cd psychodid
```

### uv

Для управления зависимостями крайне рекомендуется [установить пакетный менеджер uv](https://docs.astral.sh/uv/getting-started/installation/). 

Чтобы установить зависимости (через uv) используйте команду:

```bash
uv sync
```

### env

Установите переменные окружения. Сначала выполните команду для копирования .env файла:

```bash
cp .env.sample .env
```

Затем откройте файл `.env` и впишите на место слова `ключ` ваш API token от OpenRouter. Чтобы использовать другого провайдера поменяте переменную BASE_URL.

В лабораторных работах используется GPT-4o-mini. Для ее использования из России требуется VPN. Если использовать его неудобно - поменяйте модели в файлах ноутбуков на любые китайские. Альтернативно можно использовать российских или китайских провайдеров.


## Запуск

В вашей любимой IDE откройте эту директорию (папку) и выберите виртуальное окружение `.venv` в текущей директории. Оно создается автоматически после применения `uv sync`, описанного выше
